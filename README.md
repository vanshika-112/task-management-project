# 📝 Taskeyy - Task Management System

## About

TaskFlow is a simple task management web application built using Flask. It allows users to register, log in, and manage their personal tasks through an easy-to-use dashboard.

## Features

* User Registration & Login
* Secure Password Hashing
* Add, Edit and Delete Tasks
* Mark Tasks as Completed
* Personal Dashboard
* Task Statistics (Total, Pending & Completed)

## Tech Stack

* Python
* Flask
* SQLite
* HTML
* CSS
* JavaScript

## Installation

1. Clone the repository.
2. Install the required packages.

```bash
pip install flask
```

3. Run the application.

```bash
python app.py
```

4. Open your browser and visit:

```text
http://127.0.0.1:5000
```
